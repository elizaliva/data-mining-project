# Online Advertisement Click Prediction

## Problem Statement
The goal of this project is to predict whether an online advertisement will receive a click based on its features. This is a binary classification problem where the model determines the likelihood of user engagement with an ad.

## Target Variable
The target variable is whether an advertisement is clicked:
- 1 → Clicked
- 0 → Not Clicked

## Dataset
The dataset used in this project is the **CTR in Advertisement dataset** sourced from Kaggle.
